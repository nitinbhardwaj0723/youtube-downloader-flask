# 🎥 YouTube Downloader (Flask)

A simple and clean **YouTube Video & Audio Downloader** built using **Python Flask**, **HTML**, **CSS**, **JavaScript**, and **yt-dlp**.

This project allows users to paste a YouTube video link, fetch video details, and download either the **video** or **audio** directly from a local web interface.

## 🚀 Features

- Paste YouTube video URL
- Fetch video information instantly
- Show:
  - Video title
  - Thumbnail
  - Channel name
  - Duration
- Download:
  - 🎥 Video
  - 🎵 Audio
- Clean and simple user interface
- Runs on **localhost**
